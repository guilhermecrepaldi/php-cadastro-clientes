# Notas do Projeto - Cadastro de Clientes

## Stack
- PHP 8.2 + PDO
- MySQL
- Bootstrap 5 (CDN)

## Melhorias futuras
- Máscaras JS nos campos (CPF, telefone, CEP)
- Modal de confirmação para exclusão
- Exportar CSV
- Autenticação de usuário
- Upload de foto do cliente

## Observações
- CPF é validado tanto no servidor (PHP) quanto no banco (UNIQUE)
- Paginação de 10 registros por página
- Busca por nome OU email simultaneamente
